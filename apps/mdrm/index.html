Micro Data Reference Manual

The Micro Data Reference Manual (MDRM) is a catalog of data series included in the Board's micro data (Financial and Structure) files.

Data Dictionary
The Data Dictionary section defines each item and shows all of the reporting series on which that item appears, by date.

Reporting Forms & Mnemonics
The Reporting Forms and Mnemonics section includes a brief description of each reporting series, the series mnemonics (abbreviations), the reports, form number, and a link to the reporting form.
